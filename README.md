# Bilibili Omnipotent Shield (哔哩哔哩全能护盾)

**Railgun Ultimate Edition | Version 1.3.1-Release**
## 全能护盾能陪各位的就到这里了，代码全部开源，随意使用！
**哔哩哔哩全能护盾** 是一个集成了广告净化、功能解锁、界面美化与安全防护于一体的 Tampermonkey 用户脚本。

本项目基于 **"Railgun" 架构** 重构，旨在提供 LV6 级别的 B 站浏览体验。它不仅拥有以 **御坂美琴（Misaka Mikoto）** 为灵感的现代化 UI 设计，更在底层逻辑上实现了对页面元素的 DOM 级精准控制。

---

## 核心特性 (Level 5 Capabilities)

### 1. 超电磁炮 UI (Railgun Interface)

* **常盘台风格设计**：界面配色采用标志性的米色与电击蓝，致敬常盘台中学制服与电击使能力。
* **物理动态交互**：悬浮球采用"硬币"设计，集成物理回弹动画与呼吸效果，交互手感 Q 弹流畅。
* **Shadow DOM 隔离**：控制面板运行于独立的 Shadow DOM 中，与页面样式完全隔离，杜绝冲突。

### 2. 绝对防御 (Absolute Defense)

* **智能清道夫 (Smart Cleaner)**：基于 MutationObserver 的 DOM 级防护，精准识别并移除随机类名的广告占位符。
* **御坂网络词库 (Misaka Network)**：接入云端实时更新的屏蔽规则库，支持正则表达式与白名单机制，精准过滤引战弹幕与垃圾评论。
* **自动裁决**：可选开启"自动拉黑"功能，自动将被屏蔽规则命中的恶意用户移入黑名单。
* **弹幕力场**：实时拦截并清除命中屏蔽词的弹幕节点，还你清净的观影环境。

### 3. 能力解禁 (Feature Unlock)

* **超清画质解锁**：突破登录限制，未登录状态下也可强制开启 1080P+ 高码率画质（自动清除本地缓存以防止回退）。
* **访客评论查看**：解锁未登录状态下的评论区加载限制，允许游客自由浏览视频评论与楼中楼。
* **完整评论体验**：支持评论翻页、最热/最新排序切换、粉丝勋章显示、图片预览等完整功能。

### 4. 严谨控制 (Strict Control)

* **Opt-In 策略**：遵循"最小权限"原则，所有功能默认关闭。用户拥有绝对控制权，仅开启需要的模块。
* **登录弹窗阻断**：强力拦截播放暂停、每日访问时的登录诱导弹窗，归还清爽的浏览权。

1. 安装脚本管理器：Tampermonkey (推荐) 或 Violentmonkey。
2. 点击脚本安装链接（或将源码复制到新建脚本中）。
3. 打开 Bilibili 任意页面，屏幕右下角将出现一枚 **旋转的硬币**。
4. 点击硬币打开控制面板，根据需求勾选相应功能开关。

## 云端规则同步 (Misaka Network Sync)

脚本接入御坂网络，支持云端屏蔽词库实时同步。

### 同步机制

* **自动同步**：每 24 小时自动检查并拉取最新规则
* **手动同步**：在控制面板点击「立即更新云端词库」按钮
* **覆盖策略**：云端数据会完全覆盖本地缓存，确保规则库始终最新

### 规则优先级

```
白名单 > 黑名单 > 本地预设词
```

即使内容命中黑名单，如果同时存在于白名单中，也不会被过滤。

### 本地预设词

以下关键词为脚本内置，不受云端规则影响：

```
免费教程、实操变现、日入、加我、v信、淘宝、兼职
```

## 常见问题 (FAQ)

### Q: 评论加载失败怎么办？

A: 可能是 B 站 API 变更或网络波动，请尝试：
1. 刷新页面重试
2. 手动更新云端词库
3. 检查浏览器控制台是否有报错信息
4. 确认是否有其他脚本产生冲突

### Q: 控制面板/悬浮球消失了？

A: 点击浏览器扩展栏中的 Tampermonkey 图标，找到本脚本，点击「强制重置 UI」菜单项。

### Q: 自动拉黑功能不生效？

A: 自动拉黑功能需要登录状态下才能执行，因为需要获取 CSRF Token 来调用 B 站 API。

### Q: 如何贡献屏蔽词？

A: 请联系作者提交你的屏蔽词建议。云端规则更新后，所有用户将在下次同步时自动获取。
