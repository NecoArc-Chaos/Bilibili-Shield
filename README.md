
# 🛡️ Bilibili 全能护盾 (Bilibili Omnipotent Shield)

> **可能是目前最强力、最无感、最智能的 B 站净化脚本。**
> 基于 **API 劫持 (Fetch Hook)** 与 **DOM 拆迁** 双核驱动，从源头抹杀广告、营销号、烂梗、综艺及不想看到的分区。

## 📖 简介

受够了 B 站首页的“XX大型纪录片”烂梗？
厌倦了伪装成视频的“送AI教程”营销号？
不想在首页看到“综艺”、“课堂”或者“电视剧”楼层？
评论区全是“加群”、“领资料”的水军？

**Bilibili 全能护盾** 专为解决这些痛点而生。它不同于传统的元素隐藏脚本，它直接**接管浏览器的数据请求**，在数据渲染成画面之前，就将垃圾信息剔除，实现**真正的无感净化**。

## ✨ 核心功能

### 1. 🧬 API 核弹级净化 (Core)

脚本核心采用 `window.fetch` 劫持技术，对 B 站所有 API 响应数据进行递归扫描。

* **零容忍秒杀**：只要数据中包含“综艺”、“课堂”、“卖课”、“推广”等特征，**无视白名单**，直接从数据层面物理删除。
* **无感体验**：垃圾视频根本不会加载到你的 HTML 中，不会出现“白块”或“占位符”。

### 2. 🚫 智能黑名单系统

* **综艺/课堂毁灭者**：自动识别 API 中的 `tname` (分区) 和 `goto` (跳转类型)。凡是综艺、课堂、付费课程、游戏中心推广，统统拦截。
* **营销号杀手**：
* **名字正则查杀**：自动拉黑名字带“xx新闻”、“xx帮忙”、“xx视讯”、“xx文旅”的机构流水线账号。
* **初始号清洗**：自动屏蔽 `bili_123456` 这类未改名的尸体号/机器号。


* **卖课引流拦截**：精准识别标题中的“送教程”、“免费领”、“变现”、“实操课”等营销指纹。

### 3. 🛡️ 评论区净化

* **反烂梗**：自动过滤“乐”、“急”、“赢”、“典”等低智阴阳怪气言论（支持组合词保护，不误伤“快乐”、“双赢”）。
* **反引战**：拦截“罕见”、“1450”、“殖人”、“op”等扣帽子词汇。
* **白名单保护**：内置强大的白名单库（如“一群”、“群众”、“朱迪”），确保正常交流不被误伤。

### 4. 🏗️ 首页 DOM 拆迁队

针对 B 站首页（bilibili.com）首屏的 SSR（服务端渲染）内容：

* **楼层爆破**：自动寻找并移除“综艺”、“电影”、“电视剧”、“国创”等固定楼层容器。
* **广告去除**：暴力 CSS 隐藏所有 Banner 广告、侧栏广告和右下角浮窗。

### 5. 🎮 图形化控制台

* **即时开关**：点击页面右下角的 **🛡️** 悬浮球，即可打开控制面板。
* **功能定制**：不想杀综艺了？想看直播了？一键开关，即时生效（需刷新）。
* **状态监控**：实时显示当前拦截数量和自动拉黑人数。

## 📥 安装方法

1. **安装管理器**：首先安装 [Tampermonkey (油猴)](https://www.tampermonkey.net/) 浏览器扩展。
2. **安装脚本**：点击 [新建脚本]，将 `V1.0.1` 版本的完整代码复制粘贴进去，保存即可。
3. **开始使用**：打开 Bilibili 首页，你会看到右下角出现盾牌图标，控制台（F12）会输出 `Bilibili Shield V39 Active`。

## ⚙️ 高级配置 (User Config)

虽然脚本提供了 UI 面板，但如果你想添加**自定义的屏蔽词**或**豁免词**，可以在脚本代码顶部的 `CONFIG` 区域进行修改：

```javascript
const CONFIG = {
    // ... 其他配置 ...

    // 🟢 白名单：这里的词绝对不杀
    whiteList: [
        "我的爱豆", "特定的UP主名字", "想看的关键词"
    ],

    // ⚡ 绝杀名单：无视白名单，只要出现就杀 (慎用)
    forceKillKeywords: [
        "某某明星", "极度讨厌的词"
    ],

    // 🏗️ 楼层拆除：想看什么区就把它删掉
    targetFloors: [
        "综艺", "课堂", "推广", "直播", "娱乐" 
        // 如果想看电影，把 "电影" 从这里删掉
    ],
    
    // ...
};

```

## 🛠️ 技术原理 Q&A

**Q: 为什么有时候页面会闪一下？**
A: 这是因为 B 站首屏的部分内容是写死在 HTML 里的。脚本会用 CSS 和 DOM Observer 尽快移除它们，但浏览器渲染速度极快，偶尔可能会捕捉到一丝残影。下滑加载的内容则是 100% 无感的。

**Q: "自动拉黑" 是真的拉黑吗？**
A: 是的。当脚本检测到“剧毒”营销号（如“xx新闻”或卖课号）时，会调用 B 站的 `api.bilibili.com/x/relation/modify` 接口，执行真实的拉黑操作。这意味着即使你卸载了脚本，这些人也永远回不来了。

**Q: 会误伤我的“电子榨菜”吗？**
A: 脚本内置了“视频标题豁免词库” (`videoTitleExemptions`)。
例如：评论区发“第一”会被删（抢楼），但视频标题叫“全网第一”会被放行。
且白名单 (`whiteList`) 优先级极高，你可以把喜欢的关键词加进去作为“护身符”。
