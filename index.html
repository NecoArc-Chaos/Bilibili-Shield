安装指南

  1. 安装脚本管理器

    推荐使用 Tampermonkey (油猴) 或 Violentmonkey。

  2. 安装脚本

    点击页面右上角的「获取脚本」按钮，或将源码复制到新建脚本中。

  3. 启动御坂网络

    打开 Bilibili 任意页面，寻找屏幕右下角的 旋转硬币。点击即可开启控制面板。

数据库配置

批量拉黑功能依赖远程用户 ID 列表，您只需准备一个可公开访问的文本或 JSON 文件。

支持的格式

JSON 数组

[123456, 789012, "345678"]

纯文本（每行一个 UID）

123456
789012
345678

如何获取直链（以 GitHub Gist 为例）

  1. 登录 GitHub Gist，新建一个 Gist。
  2. 填入文件名（如 blocklist.json）和内容。
  3. 点击“Create public gist”，然后点击右上角的 Raw 按钮。
  4. 复制浏览器地址栏中的链接，即为直链。

将直链填入脚本开头的 REMOTE_BLOCK_USER_LIST_URL_PORN 或 REMOTE_BLOCK_USER_LIST_URL_CATMEME 即可使用。

注意：已拉黑的用户会永久保存在本地，不会重复拉黑。
